## 🔐 4. Data Security

### 4.1 Security Measures
- **Encryption**: TLS 1.2+ for all data in transit
- **Storage**: Secure encrypted storage for sensitive data
- **Access Control**: Role-based access controls (RBAC)
- **Audit Logging**: All access logged and monitored
- **Regular Security Reviews**: Penetration testing and vulnerability assessments

### 4.2 GitLab Token Handling
- **Local Storage Only**: Tokens stored only in your browser, never on our servers
- **No Transmission**: Tokens never transmitted to our backend
- **Read-Only**: We recommend using read-only tokens
- **User Control**: You control token scope and expiration

### 4.3 What We DON'T Store
- ❌ GitLab API tokens
- ❌ Source code or raw repository content
- ❌ Your GitLab credentials
- ❌ Unencrypted security data
- ❌ Personally identifiable information (unless authorized)
